# Process campus-level enrollment data

Combines race/ethnicity and gender data at the school level.

## Usage

``` r
process_campus_enr(race_df, gender_df, end_year)
```

## Arguments

- race_df:

  Raw race/ethnicity data frame

- gender_df:

  Raw gender data frame

- end_year:

  School year end

## Value

Processed campus data frame
