# Download raw enrollment data for Era 0 (2006-2010)

Historical data from South Dakota DOE. These files use XLS format with
varying structures. 2007 only has summary totals, while other years have
grade-level breakdowns.

## Usage

``` r
get_raw_enr_era0(end_year)
```

## Arguments

- end_year:

  School year end

## Value

List with district and campus data frames
