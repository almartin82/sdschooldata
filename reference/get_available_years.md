# Get available years for SD enrollment data

Returns the range of years available for download from the South Dakota
Department of Education.

## Usage

``` r
get_available_years()
```

## Value

Integer vector of available years

## Examples

``` r
get_available_years()
#>  [1] 2006 2007 2008 2009 2010 2011 2012 2013 2014 2015 2016 2017 2018 2019 2020
#> [16] 2021 2022 2023 2024 2025
```
